# RFC #046: Data Range Display (Begin/End)

## Description

Implement data range display showing the beginning and end of data values similar to xarray's native representation, providing users with quick data overview information.

## Requirements

- Display data range for each variable
- Show beginning and end values
- Support different data types
- Handle large datasets efficiently
- Provide data type information
- Support both dataset and datatree
- Display in user-friendly format
- Support data sampling for large ranges

## Acceptance Criteria

- [ ] Data ranges are displayed
- [ ] Beginning and end values are shown
- [ ] Different data types are supported
- [ ] Large datasets are handled efficiently
- [ ] Data type information is clear
- [ ] Both dataset and datatree work
- [ ] Display format is user-friendly
- [ ] Data sampling works for large ranges
