# RFC #037: Categorical Raster Plotting

## Description

Implement categorical raster plotting functionality using NetCDF conventions for flag meanings and categorical data visualization in the Scientific Data Viewer.

## Requirements

- Support categorical data plotting
- Use NetCDF conventions for flag meanings
- Implement appropriate color schemes for categories
- Handle flag value mappings
- Support legend generation for categories
- Provide categorical data statistics
- Support both discrete and continuous categorical data
- Integrate with existing plotting system

## Acceptance Criteria

- [ ] Categorical data is properly identified
- [ ] Flag meanings are displayed correctly
- [ ] Appropriate color schemes are used
- [ ] Legends show category information
- [ ] Statistics are calculated for categories
- [ ] Integration with existing plots works
- [ ] Performance is acceptable
- [ ] User can customize categorical plotting
