# RFC #044: Data Structure Collapse/Expand Controls

## Description

Implement collapse all and expand all controls for the Data Structure view to improve navigation and user experience when working with large, complex datasets.

## Requirements

- Add "Collapse All" button
- Add "Expand All" button
- Implement state management for collapse/expand
- Support keyboard shortcuts
- Provide visual indicators for state
- Maintain user preferences
- Support selective collapse/expand
- Handle large datasets efficiently

## Acceptance Criteria

- [ ] Collapse All button is functional
- [ ] Expand All button is functional
- [ ] State is properly managed
- [ ] Keyboard shortcuts work
- [ ] Visual indicators are clear
- [ ] User preferences are saved
- [ ] Selective controls work
- [ ] Performance is acceptable
