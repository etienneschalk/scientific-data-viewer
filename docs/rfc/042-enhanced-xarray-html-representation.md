# RFC #042: Enhanced Xarray HTML Representation

## Description

Align the Scientific Data Viewer's HTML representation with xarray's native HTML repr, including the ability to show attributes and partial data previews for better data exploration.

## Requirements

- Implement xarray-style HTML representation
- Show variable and coordinate attributes
- Display partial data previews
- Support collapsible sections
- Maintain xarray visual consistency
- Support both dataset and datatree
- Provide data type information
- Support attribute filtering

## Acceptance Criteria

- [ ] HTML representation matches xarray style
- [ ] Attributes are displayed appropriately
- [ ] Data previews are shown
- [ ] Collapsible sections work
- [ ] Visual consistency is maintained
- [ ] Both dataset and datatree are supported
- [ ] Data types are clearly indicated
- [ ] Attribute filtering is available
